Function implementation

function F = adaptivefilt_base(I, H, M)

dealing with multispectral images

C = size(I,3);

if C>1
    F = I;
    for c=1:C
        F(:,:,c) = adaptivefilt_base(I(:,:,c), H, M);
    end
    return;
end

call the mex file

F = adaptivefilt_mex(I, H, M);

end % end of adaptivefilt_base
