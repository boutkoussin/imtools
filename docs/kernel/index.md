### `kernel` - Kernel operators
 
* [gausskernel](gausskernel.html)     - Compute a 1D or 2D Gaussian filter.
* [dirgausskernel](dirgausskernel.html) - Compute an oriented Gaussian kernel.
* [hourglasskernel](hourglasskernel.html) - Compute an oriented non-linear spatial hourglass filter.
* [euclidkernel](euclidkernel.html)   - Create a kernel spatially weighted with the euclidean distance.
* [local3x3kernel](local3x3kernel.html) - Compute local kernels defined by zones and levels.
* [neiposkernel](neiposkernel.html)   - Create the kernel containing the indices and subscript positions of the grid pixels of a given square window.
